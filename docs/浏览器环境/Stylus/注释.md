# 注释

Stylus支持三种注释，单行注释，多行注释，以及多行缓冲注释

## 单行注释

跟JavaScript一样，双斜杠，CSS中不输出。
```
// 我是注释!
body
    padding 5px // 蛋疼的padding
```

##多行注释

多行注释看起来有点像CSS的常规注释。然而，它们只有在compress选项未启用的时候才会被输出。
```
/*
 * 给定数值合体
 */

add(a, b)
    a + b
```

## 多行缓冲注释

跟多行注释类似，不同之处在于开始的时候，这里是`/*!. `这个相当于告诉Stylus压缩的时候这段无视直接输出。
```
/*!
 * 给定数值合体
 */

add(a, b)
    a + b
```
